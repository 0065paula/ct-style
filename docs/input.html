Input

Displays a form input field or a component that looks like an input field.

Installation

bash
npx shadcn@latest add input --registry https://your-registry-url

Input - Default

tsx
import { Input } from "@/components/ui/input" export function InputDemo() { return ( <> <Input placeholder="Enter text..." /> <Input value="Focused" /> </> ) }

Input - Sizes

tsx
import { Input } from "@/components/ui/input" export function InputSizes() { return ( <> <Input size="sm" placeholder="Small" /> <Input size="default" placeholder="Default (Medium)" /> <Input size="lg" placeholder="Large" /> </> ) }

Input - Error State

Please enter a valid email
tsx
import { Input } from "@/components/ui/input" import { Label } from "@/components/ui/label" export function InputError() { return ( <div className="grid w-full max-w-sm items-center gap-1.5"> <Label htmlFor="email">Email</Label> <Input id="email" type="email" variant="error" placeholder="Email" /> <p className="text-sm text-destructive">Please enter a valid email</p> </div> ) }

Input - Disabled

tsx
import { Input } from "@/components/ui/input" export function InputDisabled() { return <Input disabled type="email" placeholder="Email" /> }

API

Prop Type Default Description
size "sm" | "default" | "lg" "default" The size of the input
variant "default" | "error" "default" The visual style variant of the input
type string "text" The type of input. All standard HTML input types are supported
className string - Additional CSS classes
...props React.InputHTMLAttributes - All standard input HTML attributes

Card

Displays a card with header, content, and footer.

Installation

bash
npx shadcn@latest add card --registry https://your-registry-url

Usage

Card Title

Card Description

Card Content

Card Footer

tsx
import { Card, CardContent, CardDescription, CardFooter, CardHeader, CardTitle, } from "@/components/ui/card" export function CardDemo() { return ( <Card> <CardHeader> <CardTitle>Card Title</CardTitle> <CardDescription>Card Description</CardDescription> </CardHeader> <CardContent> <p>Card Content</p> </CardContent> <CardFooter> <p>Card Footer</p> </CardFooter> </Card> ) }

With Button

Create project

Deploy your new project in one-click.

Card content goes here.

tsx
import { Card, CardContent, CardDescription, CardFooter, CardHeader, CardTitle, } from "@/components/ui/card" import { Button } from "@/components/ui/button" export function CardWithButton() { return ( <Card className="w-[350px]"> <CardHeader> <CardTitle>Create project</CardTitle> <CardDescription>Deploy your new project in one-click.</CardDescription> </CardHeader> <CardContent> <form> <div className="grid w-full items-center gap-4"> <div className="flex flex-col space-y-1.5"> <Label htmlFor="name">Name</Label> <Input id="name" placeholder="Name of your project" /> </div> </div> </form> </CardContent> <CardFooter className="flex justify-between"> <Button variant="outline">Cancel</Button> <Button>Deploy</Button> </CardFooter> </Card> ) }

API

Component Props Description
Card React.HTMLAttributes<HTMLDivElement> Main card container
CardHeader React.HTMLAttributes<HTMLDivElement> Container for the card header section
CardTitle React.HTMLAttributes<HTMLHeadingElement> The card title component (renders as h3)
CardDescription React.HTMLAttributes<HTMLParagraphElement> The card description component
CardContent React.HTMLAttributes<HTMLDivElement> Container for the main card content
CardFooter React.HTMLAttributes<HTMLDivElement> Container for the card footer section
